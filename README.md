# Grey Hack PTR P

This repository contains a focused subset of scripts extracted from the broader Grey Hack workspace.

## Included content

- Standalone GreyScript utilities and operator helpers from the root set
- `src/` modular CRS-related sources and variants
- `pw/` password and escalation-related helpers
- `stealth/` stealth-oriented helper scripts

## Notes

- This repo intentionally excludes the larger documentation, tooling, archives, and vendor/reference folders from the main workspace.
- File names and layout are kept close to the original workspace to preserve compatibility with existing build or in-game usage patterns.

## Purpose

Use this repository as the smaller, portable split of the main Grey Hack project when you only want the selected script set.
